| What Is This Project?

This is a modern, open-source reboot of the classic USB Rubber Ducky — but turbocharged for the ESP32. At its core, it turns your ESP32-based development board (like the LilyGO T-Dongle S3) into a powerful, scriptable USB attack platform that supports complex logic, wireless management, and live editing via a self-hosted web interface.

Instead of requiring compilation or host software, this firmware hosts its own full web interface on the device — just power it on, connect over Wi-Fi, and drop in your payloads.

| Who Is It For?

This project is aimed at:

  • Penetration testers & red teamers who need powerful USB HID scripting with stealth
  • Developers who want to automate USB keyboard/mouse input, clipboard actions, and more
  • Makers & educators teaching hardware hacking, browser abuse, or keyboard injection
  • ESP32 enthusiasts who want to experiment with HID/BLE and embedded UI

| Features at a Glance

  • ✅ Supports full DuckyScript v1–v3 syntax
  • ✅ Built-in variables and flow control (`DEFINE`, `IF`, `WHILE`, etc.)
  • ✅ Web-based script editor with real-time execution
  • ✅ USB HID keyboard/mouse injector
  • ✅ BLE and serial emulation (optional modes)
  • ✅ Optional TFT display feedback
  • ✅ Device-bound license generation (via MAC/UID)
  • ✅ Full local file system (via SPIFFS)

| Under the Hood

The firmware is built with ESP-IDF and includes:

  • Custom license key generation (based on MAC, flash UID)
  • Formatter modules for generating 5-part keys
  • SPIFFS filesystem for payload storage
  • ESP USB Host Stack (for HID injection)
  • Web server (ESPAsyncWebServer or Mongoose)
  • Debugging over JTAG + GDB via OpenOCD
